# Salesforce Environment Setup: Playground Management
A Trailhead Playground is a dedicated Salesforce environment used for learning, testing, and development without affecting live business data.
## 1. Creating the Trailhead Playground
* Accessed the Trailhead module and selected Create Playground.
* Waited for the Salesforce org setup to complete.
* Verified successful creation by launching the playground through the Launch button.
## 2. Managing Playground Access
* Opened the Playground Starter app to retrieve login credentials.
* Accessed the Get Your Login Credentials section to view the username.
* Reset the password using the Reset My Password option.
* Configured secure credentials for future logins and development activities.
## 3. Installing Apps and Packages
* Navigated to the Install a Package section in the Playground Starter app.
* Entered the provided package ID beginning with 04t.
* Installed the package using the Install for Admins Only option.
* Verified successful installation through:
* Setup → Installed Packages
## 4. CRM Practice Activities
* Created customer accounts and contact records.
* Created and managed sales opportunities.
* Explored Salesforce Lightning Experience features such as:
  * App Launcher
  * Navigation Bar
  * Opportunity Workspace
 ## 5. Skills and Concepts Learned
* Understanding the difference between a Trailhead Playground and a Developer Edition org.
* Navigating Salesforce Lightning Experience.
* Managing Salesforce credentials and environment security.
* Installing and verifying managed packages.
* Working with Salesforce CRM objects such as Accounts, Contacts, and Opportunities.
